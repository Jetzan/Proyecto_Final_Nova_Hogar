# Proyecto_Final_Nova_Hogar
Nova Hogar es una página web diseñada para la venta en línea de productos para el hogar, ofreciendo a los usuarios una experiencia moderna, intuitiva y segura. El objetivo del proyecto es facilitar la compra de artículos de uso doméstico mediante una interfaz sencilla, categorías organizadas y un sistema de carrito de compras eficiente.
